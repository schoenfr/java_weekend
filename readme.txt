Run the backend first:

- Import project into IntelliJ (use file pom.xml for importing the project)
- Navigate to class LibraryApp (via src/main/java/...)
- Right click class and select Debug 'LibraryApp.main()'
- Try out the backend by opening http://localhost:8080/api/books

Then run the frontend:

- Use a terminal and "run npm install"
- Then run "npm start"
